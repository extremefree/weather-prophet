# Weather Prophet — 最终测试报告

## 模型版本
- 优化器: Adam (自适应梯度) + 平滑 Laplace 先验 (Huber近似, ε=τ×0.5)
- 数据标准化: absmax scaling (与官方 Prophet 一致)
- 季节性: Fourier 级数 (yearly order=10, weekly order=3, period=365.25)
- 增长类型: LINEAR, LOGISTIC, FLAT
- 季节性模式: ADDITIVE, MULTIPLICATIVE, **MIXED** (新增)

## MIXED 季节性模式 (新增)

MIXED 模式同时生成加法和乘法傅里叶特征:
- 加法部分: `y += X_sa × β_sa` — 捕捉恒定振幅的周期信号
- 乘法部分: `y += trend × X_sm × β_sm` — 捕捉振幅随趋势增长的周期信号

公式: `y(t) = trend(t) × (1 + X_sm(t) × β_m) + X_sa(t) × β_a`

这使得模型能够捕捉「振幅随时间增长/缩小」的周期现象, 如:
- 温度波动幅度逐年增大 (全球变暖效应)
- 销售额季节性波动随业务增长而放大
- 信号强度随系统增益变化

## 为什么 R² 是误导性指标

R² = 1 - SS_res / SS_tot。在 60 天短窗口 + 高噪声 (σ≥2) 条件下:

1. **信号方差小**: 60天窗口可能只覆盖季节性信号的一小段 (如冬季→春季), SS_tot 较小
2. **噪声方差大**: 即使模型完美预测了期望值, 残差仍包含噪声 (σ²), SS_res 不为零
3. **R² 膺失**: R² ≈ signal_var / (signal_var + noise_var), 当 noise_var >> signal_var 时, R² 必然很小

**MAE/σ 才是正确指标**: 它直接衡量「预测误差是否低于噪声水平」。MAE/σ < 1.0 = 模型捕捉了信号。

## 关键技术改动

### 1. MIXED 季节性模式 (解决振幅增长问题)
```
y(t) = trend(t) × (1 + X_mult(t) × β_m) + X_add(t) × β_a
```
- 加法特征 (β_a): 捕捉恒定振幅的周期成分
- 乘法特征 (β_m): 振幅随趋势缩放, 解决振幅增长场景

AmpGrowth 场景从 **MAE/σ=1.23 (FAIL)** 改善为 **MAE/σ=0.63 (PASS)**。

### 2. Adam 优化器 (替代纯 L-BFGS)
- 自适应学习率: 每参数独立调整, 适应不同梯度尺度
- 学习率调度: lr=5e-4 → 5e-3 → 5e-4 (探索→加速→精细收敛)
- 停滞检测 + 重置: 防止陷入局部振荡

### 3. 平滑 Laplace 先验
- Huber近似: `sqrt(x² + ε²)` 替代 `|x|`, ε=τ×0.5
- 消除梯度不连续性, 使 Adam 更稳定
